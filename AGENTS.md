## target
develop a picgo plugin for using MarSeventh/CloudFlare-ImgBed as a uploader

## main function
- rename the uploading image like "yyyymmddHHmmssSSS"
- then, upload the renamed image to an existing instance of MarSeventh/CloudFlare-ImgBed

## source documents
- api document for MarSeventh/CloudFlare-ImgBed in: ./reference/Cloudflare-Imgbed-Doc
- picgo plugin developing priciples in: ./reference/PicGo-Core-Doc

## other details you should know
- the plugin is only for picgo-cli now, not consider using for picgo-gui.
- try to be simple.
